# Bash-DBMS-GUI
## A simulating Database Management System using Bash and Graphical User Interface
You can Run this project by running first.sh file

## Features For Database
- Create Database
- List Databases
- Rename Database
- Drop Database
- Connect to Database

## Features For Tables
- Create Table 
- List Tables
- Rename Table
- Insert Into Table
- Select From Table
  * Select All
  * Select a Row
  * Select a column
- Delete from table
  * Delete row
  * Delete contents of table
  * Delete contents of a column
- Update Table
  * Update value in a Row
  * Update Multiple Values in a Column
